# Jura
식당 주문 / 랜덤 메뉴 추천

식당에 가기전 미리 주문할 수 있고, 지역에 따라 랜덤으로 메뉴를 추천해주는 웹

## 주요 기능

- 지역을 선택하고, 한식, 일식, 중식, 양식 등 나눠 식당을 선택 할 수 있습니다.
- 랜덤 메뉴 추천(지역 선택 후 점심, 저녁, 그냥(점심,저녁포함) 을 선택하면 그에 맞는 랜덤 메뉴를 처찬해줍니다)

## 개발 환경 & 언어

JAVA / JavaScript / Jsp / html

Eclipse

Tomcat 9.0

Oracle
